Personal Tool

Simulation Project

Flight Sim

A browser flight simulator built inside Gremlin Hub that grew from a lightweight prototype into a proper 3D flying sandbox. The focus is on readable runway approaches, believable control feel, and the kind of short landing practice loop that is fun to keep tuning.

  • Custom WebGL-style 3D scene with aircraft, runway, and terrain rendering
  • Iterated around fixed-wing handling, camera behavior, trim, and landing feel
  • Validated with repeatable scripted checks for flight, crash, and landing behavior
Open Flight Sim

Arcade Game

Block Blast

A browser puzzle game inside Gremlin Hub built around the satisfying loop of placing pieces, clearing rows and columns, and keeping the board alive as long as possible. It takes the feel of a mobile block puzzle and turns it into a polished desktop-and-phone web version.

  • 8x8 endless puzzle board with drag-and-place and click placement
  • Row and column clears, combo scoring, fullscreen mode, and seeded opening racks
  • Designed and tuned as a lightweight game experiment inside the larger Gremlin Hub system
Open Block Blast

Meal Planning Tool

Receipt Meal Planner

A grocery receipt analysis tool inside Gremlin Hub that turns a store receipt into a usable meal planning pass. It extracts receipt text, identifies what was bought, infers ingredients, and turns the haul into realistic recipe ideas without manually typing everything in.

  • OCR-first workflow tuned for tall grocery receipt photos and screenshots
  • Uses local model pipelines with a faster Codex CLI fallback for analysis
  • Saves dated analyses with parsed items, inferred ingredients, and recipe suggestions

Archive Project

Childhood Timeline

A timeline project inside Gremlin Hub that reconstructs childhood memories and family history into a browsable narrative view. Instead of manually writing everything from scratch, it turns archived social media history into something easier to scan, summarize, and revisit.

  • Builds a structured timeline from captured family and childhood posts
  • Data was acquired by parsing Facebook feed/archive HAR captures into generated timeline data
  • Combines text extraction, filtering, tagging, and summary generation inside Gremlin Hub
