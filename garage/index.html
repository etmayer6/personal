How it works

A real workflow, safely staged.

The original Gremlin Hub tool tracks issues for a private vehicle. This version preserves the product thinking while replacing identifying information with a fictional machine and simulated telemetry.

All edits stay in localStorage. The schematic, scan sequence, filters, export, and service queue are dependency-free so the demo remains fast on a static site.
